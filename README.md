# Project

Situs web statis sederhana berisi HTML, CSS, dan JavaScript.

## Deskripsi

Proyek ini berisi halaman statis untuk tujuan demo atau portofolio. Struktur proyek sudah termasuk komponen header/footer, aset gambar, CSS responsif, dan skrip JS kecil.

## Struktur Direktori (ringkas)

- `index.html` - Halaman utama
- `standar.html` - Halaman tambahan
- `assets/` - Aset proyek
- `components/` - Komponen HTML (header, footer)
- `css/` - File CSS (`style.css`, `responsive.css`)
- `js/` - Skrip JavaScript (`script.js`, `darkmode.js`, `counter.js`)

## Cara Menjalankan

1. Buka `index.html` di browser (double-click atau `Open with Live Server`).
2. Tidak diperlukan server jika hanya file statis; gunakan Live Server untuk pengembangan lebih nyaman.

## Pengembangan

- Edit file di folder `components/` untuk mengubah header/footer.
- Edit file CSS di `css/` untuk mengubah tampilan.
- Skrip JavaScript ada di `js/`.
